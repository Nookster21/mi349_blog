What is a ROM Hack?

To start off, ROM hacking is not actually hacking but it's editing. When ROM hacking, you edit the ROM files to be able to change features of a video game. Some examples of those features would be dialogue, graphics, maps, and storyline.

Methods to ROM Hacks

  • Hex Editing: Mainly used to edit texts.
  • Graphics Editing: Changing the look of the video game's environments, characters, fonts, and so much more.
  • Palette Editing: Changes the color values in the game.
  • Data Editing: Changes the data of characters, items, and possible enemies.
  • Assembly Hacking: Most powerful and difficuly hacking technique. The official process is called ASM hacking, which is editing the game's actual code.
  • Music Hacking: One of the most rarest hack to use because it's difficult to find music data to modify it, and porting music from another game.
  • ROM Expansion: This increases the ROM image so there would be more room for content to make a larger game.

    • There are different types of methods that are used to create a ROM hacks. If you're interested, I recommend going to ROM hack forums to learn more about them.

Systems and Games

Majority of systems that are ROM hacked are NES and SNES games because they are simple to edit compared to recent consoles(does not mean they can not be hacked). Many hacks have been released using the Sonic the hedgehog series, Super Mario series, Pokemon series, and many others.
