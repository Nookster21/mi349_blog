What is a ROM Hack?

To start off, ROM hacking is not actually hacking but it's editing. When ROM hacking, you edit the ROM files to be able to change features of a video game. Some examples of those features would be dialogue, graphics, maps, and storyline.

Methods to ROM Hacks

  • Hex Editing: Mainly used to edit texts.
  • Graphics Editing: Changing the look of the video game's environments, characters, fonts, and so much more.
  • Palette Editing: Changes the color values in the game.
  • Data Editing: Changes the data of characters, items, and possible enemies.

There are so much more types of methods that are used to create a ROM hacks. If you're interested, I recommend going to ROM hack forums to learn more about them.
